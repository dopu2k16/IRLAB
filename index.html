ABOUT

At IR-Lab, Dept. of CSE, IIT Kanpur, we are interested in Information Retrieval, Natural Language Processing, Data Mining,and Machine Learning.
Our current focus in on cross-lingual question answering and multilingual topic modeling. We aim to build language independent models for cross-lingual tasks.

PROJECTS

Multilingual Embeddings for Cross-Lingual IR in absence of aligned corpora

Natural Language Processing, Neural Networks and Information Retrieval
- Developed an algorithm using merge and shuffle techniques to create multilingual word embeddings based on Word2vec/Glove model -IR task of retrieving documents in one language(eg., Hindi) from a mixed corpus using queries of another language(eg., English) without using parallel aligned data or an external dictionary
- Developing a search engine framework for Indo-Aryan languages

Cross-Lingual question answering
Without using language dependent sources such as dictionaries, chunkers, etc., we are trying to build a language independent cross-lingual question answering framework.

Multilingual topic modeling
We are trying to develop a multilingual topic modeling framework in absence of aligned corpus or parallel corpus. This framework will cluster multilingual words in topics clusters.
